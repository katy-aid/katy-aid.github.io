# Katy Aid

A collection of resources available in the community of Katy.

## Local Development

### Prerequisites

- A modern web browser

- A code editor

### How to Run

1. Open the terminal

2. Clone this repository with Git:

    ```bash
    git clone https://github.com/katy-aid/katy-aid.github.io.git
    cd katy-aid.github.io
    ```
3. Open your file explorer

4. Navigate to the `katy-aid.github.io` directory

5. Open `index.html` in your browser
